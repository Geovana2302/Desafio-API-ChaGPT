# Desafio-API-ChaGPT
# 🤖 Chat IA com Flask + OpenRouter

Projeto desenvolvido em Python utilizando Flask integrado com a API da OpenRouter para criação de um chatbot inteligente estilo ChatGPT.

---

# 🚀 Tecnologias Utilizadas

- Python 3.11+
- Flask
- OpenAI SDK
- OpenRouter API
- HTML5
- CSS3
- JavaScript
- Python Dotenv
